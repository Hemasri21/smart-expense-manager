# 💰 Smart Expense Manager

A Python-based command-line application to track expenses, generate reports, and manage budgets.

## Features
- Add, view, and delete expenses
- Monthly and category-wise reports
- Budget tracking with alerts
- SQLite database integration
- Clean CLI interface using Rich library

## Tech Stack
- Python
- SQLite
- Rich

## ▶️ How to Run
```bash
python main.py
